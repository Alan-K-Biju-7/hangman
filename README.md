# Hangman Simulator (Explainable / Visual)

A clean, interactive Hangman game built with HTML/CSS/JS for learning:
- wrong guesses → hangman drawing
- lives bar + wrong list
- score + streak (saved)
- hint costs 1 life
- keyboard shortcuts + on-screen keyboard
- categories + custom word mode

## Run
Open `index.html` directly in browser, or use VS Code Live Server.

## Controls
- Click letters OR type A–Z
- Enter: New game
- H: Hint (costs 1 life)
- Backspace: Reveal
- Use Category + Difficulty dropdowns

## Files
- `index.html` UI
- `styles.css` theme + layout
- `app.js` game logic
